# PetLog 首頁快速概覽功能 PRD

## 目標
讓使用者在登入後的首頁能夠一眼掌握帳號下所有寵物的整體健康管理狀況，提升使用體驗與資訊透明度。

## 功能描述
- 在首頁顯示「快速概覽」區塊，內容包含：
  1. 寵物數量：顯示目前帳號下已建立的寵物總數
  2. 健康記錄數量：顯示所有寵物的健康記錄總數
  3. 醫療記錄數量：顯示所有寵物的醫療記錄總數
- 指標皆以明顯的數字方式呈現，方便快速瀏覽
- 僅顯示與目前登入帳號相關的資料
- 若尚無資料，顯示為 0

## 使用者故事
- 作為一位飼主，我希望在登入後首頁能快速看到我目前管理的寵物數量、健康記錄與醫療記錄總數，方便我掌握全局。

## 未來擴充方向
- 可加入「最近新增的健康記錄」、「即將到期的疫苗」等提醒
- 支援圖表化呈現（如折線圖、圓餅圖等）
- 可點擊指標進入對應詳細頁面

## 非功能性需求
- 目前暫無查詢時間限制，未來可依實際需求優化查詢效能
- UI 需支援行動裝置與桌面裝置自適應
